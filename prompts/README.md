# PROMPTS GOVERNANCE

## Rules

1. All prompts must be versioned.
2. Prompts must be short and explicit.
3. Output must be structured (JSON if applicable).
4. No hidden business logic inside prompts.
5. Prompts cannot override backend rules.

## Folder Convention

prompts/
 ├─ classify_intent_v1.md
 ├─ generate_reply_v1.md
 └─ summarize_v1.md
